# bootstrap
contains files used during the installation and configuration process of my pc

### How to

##### Linux
```bash
$ git clone -b master --depth 1 --single-branch https://github.com/simone98dm/bootstrap.git
$ cd bootstrap/
$ chmod +x *.sh
$ sudo ./full-installation.sh
$ ./vscode-extesions.sh
```
##### Windows
```powershell
$ git clone -b master --depth 1 --single-branch https://github.com/simone98dm/bootstrap.git
$ cd bootstrap/
$ *doble click on .cmd*
```
